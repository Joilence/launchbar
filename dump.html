Dump Item

Download signed Dump Item.lbaction

Dumps contents of passed in item back as LaunchBar results. Useful for LaunchBar action developers to see the environment and data being passed around.

Shows:

  • LaunchBar environment variables
  • Action environment variables
  • Handler method called
  • Arguments passed to handler method

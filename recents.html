Download signed Recents.lbaction

Action that runs a Finder/Spotlight Saved Search/Smart Folder and returns the results to LaunchBar. Used most commonly with a Saved Search that returns recent (created, opened, modified) files and folders.

  1. Create your Saved Search in Finder, name it "Recents" and save it. See this example
  2. Run the action, so the default preferences get set
  3. Open the action preferences file ~/Library/Application Support/LaunchBar/Action Support/com.renaghan.launchbar.Recents and edit the find command to your liking
  4. See the man page for mdfind for more information 
        
        
Supports Action Updates framework and action

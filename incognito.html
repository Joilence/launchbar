Incognito Browser

Download signed Incognito.lbaction

LaunchBar action that opens a Google Chrome browser window/tab in incognito private browsing mode.

Opens to:

  • URL from active Safari tab
  • URL from clipboard
  • Google Search URL with clipboard contents
  • Blank

If more than one option is available the action will show LaunchBar items for you to select from.

        
        
Supports Action Updates framework and action

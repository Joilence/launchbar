Action Updates

Download signed Updates.lbaction

Action that checks custom actions to check to see if a new version is available. Actions must specify a /LBDescription/LBUpdate key in the Info.plist of their action pointing at the URL which returns the Info.plist of the latest version of the action. Additionally they may specify a /LBDescription/LBDownload key specifying the URL to download the latest .lbaction package of the action.

If the action provides a URL to download the latest version in /LBDescription/LBDownload, then this action will attempt to download the latest version to ~/Downloads

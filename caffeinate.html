Caffeinate

Download signed Caffeinate.lbaction

Keep your system awake using built-in /usr/bin/caffeinate. Give the action a string argument in the form of 2h or 45m or simply 2 (default is hours) to set the awake duration, or 0 to stay awake forever. Type enter without an argument on the action to see current caffeinate status and the option to stop the current sleeplessness.

Dismiss Notifications

Download signed Dismiss Notifications.lbaction

Action that will dismiss all notifications from Notification Center. Will "Snooze" calendar items for conference calls (identified by having at least 14 numbers in the Location field) and will "Close" all others.

Supports Action Updates framework and action

Thanks to UI Browser for helping me figure out how to script the notification elements

Original idea from Matt Gemmell's Keyboard Maestro macro

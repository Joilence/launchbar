Switch Audio

Download signed Switch Audio.lbaction

Action that lets you switch Input and Output Audio devices. Requires switchaudio-osx which is most easily installed via homebrew brew install switchaudio-osx (command must be installed to /usr/local/bin/SwitchAudioSource).

When invoked it returns items for each output device (headphones) and each input device (microphone). The currently selected item for each is prefixed with *. Select an item and hit enter to switch to that item, the action will respond by refreshing the items and you should note the * by the item you just selected.

Supports Action Updates framework and action

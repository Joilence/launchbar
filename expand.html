LaunchBar Actions

Expand URL

Download signed Expand URL.lbaction

Action that takes a string URL as input and expands it by resolving all redirects and returning the final result URL back to LaunchBar. Useful for resolving t.co and bit.ly ... type "short" URLs into real URLs.

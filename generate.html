Generate Password

Download signed Generate Password.lbaction

Action that generates random passwords. Adjust settings via Action preferences. By default generates 5 word-based 4 word passwords using hsxkpasswd and one 15 character long password with at least 1 number and 1 symbol and avoids numbers/letters that look similar to the eye. Copies first generated password to clipboard.

        
        
Supports Action Updates framework and action

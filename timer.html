Timer

Download signed Timer.lbaction

Action that takes an input like Turn off oven 15m and creates a timer in LaunchBar. After the delay time has passed LaunchBar will show you your reminder message in a large type window with an accompanying sound.

        
        
Supports Action Updates framework and action
